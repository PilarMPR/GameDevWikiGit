# 🎭 Pleasures & Aesthetics — The Same List

LeBlanc's 8 aesthetics (MDA, s011) and Schell's 8 pleasures (s005, ch.8) are **functionally equivalent** — the same taxonomy of player motivations, independently formulated.

## The equivalence

| MDA Aesthetic (s011) | Schell Pleasure (s005) | The motivation |
|---|---|---|
| Sensation | Sensation | Sensory pleasure |
| Fantasy | Fantasy | Make-believe, escapism |
| Narrative | Narrative | Drama, story |
| Challenge | Challenge | Mastery, competition |
| Fellowship | Fellowship | Social connection |
| Discovery | Discovery | Exploration, novelty |
| Expression | Expression | Self-expression, creativity |
| Submission | Submission | Relaxation, pastime |

Both provide a vocabulary for targeting specific player experiences instead of the vague "fun."

## Why this matters

If you design around one list, you're using both. The vocabulary difference doesn't change the underlying design implications.

Bartle's four types (s005, ch.8; s001) map onto subsets of these aesthetics:
- Achievers → Challenge
- Explorers → Discovery
- Socializers → Fellowship
- Killers → Challenge + Expression
