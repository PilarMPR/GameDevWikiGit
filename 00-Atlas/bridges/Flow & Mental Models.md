# 🌊 Flow & Mental Models — Four Angles on the Same Zone

Four sources describe the same phenomenon — the optimal zone of cognitive/emotional engagement — from four different angles:

## The four framings

| Source | Framing | Emphasis |
|---|---|---|
| Csikszentmihalyi (via Schell s005) | Flow channel: challenge = skill | Phenomenological: what the zone *feels like* |
| Koster (s016) | Fun = learning; learnable middle | Mechanistic: *why* the zone is pleasurable (pattern acquisition) |
| Sellers (s001) | Engaged attention: appropriate arousal + mental model updating | Systems: the game+player coupling that produces flow |
| Hodent (s014) | Game UX: usability enables the zone; cognitive load breaks it | Cognitive science: the brain's constraints |

## The convergence

All four agree: engagement requires a challenge level that matches the player's current skill/comprehension, with clear feedback and continuous forward movement. Too easy = boredom. Too hard = frustration. The zone in between is where the magic happens.

## Mental models

The player's mental model is the player-side of the engagement equation:
- Norman (s015): conceptual model — the user's understanding of how the system works
- Sellers (s001): player mental model loop — builds continuously through play
- Schell (s005, ch.9): modeling = one of the four mental abilities; games are "pre-digested models"
