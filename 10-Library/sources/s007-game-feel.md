# s007 · Game Feel (Swink)

**TL;DR** — The first systematic treatment of "game feel" — the tactile, kinesthetic sensation of controlling a virtual object in real time. Turns an intuitive quality into something analyzable and designable, grounded in perceptual psychology (s007).
**Provenance caveats** — Summarized from chapter summaries + canonical content.

## Core framework

- **Game feel = real-time control of virtual objects in a simulated space, with interactions emphasized by polish** (s007).
- **Three building blocks:** real-time control (continuous responsive input), simulated space (spatial physics), polish (effects — sound, particles, screen shake, animation) (s007).
- **Metrics** — game feel can be *measured*: input response, position/velocity/acceleration curves. Swink dissects real games' control metrics.
- Grounds in **perceptual psychology** — proprioception, body schema, players extending sense of self into avatar.
