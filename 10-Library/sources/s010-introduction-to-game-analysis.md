# s010 · Introduction to Game Analysis (Fernández-Vara)

**TL;DR** — A method for analyzing games as cultural texts — how to study and write about an existing game rigorously. Complements the design books: where they teach you to *build*, this teaches you to *read* games critically (s010, ch.1).
**Provenance caveats** — Game studies/analysis, not a design manual. Verified TOC.

## Key framework

- **Three areas of analysis:**
  1. **Context** (ch.3) — production, genre, technological/historical/economic context, audience.
  2. **Game Overview** (ch.4) — players, rules/goals/modes, mechanics, spaces, fictional world.
  3. **Formal Elements** (ch.5) — rules, abstraction, procedural rhetoric, dynamics, difficulty.
- **Procedural rhetoric** — games make arguments through their rules and systems (s010, ch.5).
- A structured framework (Context / Overview / Formal Elements) for disciplined analysis.
