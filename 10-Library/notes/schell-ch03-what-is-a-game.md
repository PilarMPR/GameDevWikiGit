# What Is a Game? — Toy → Play → Game (s005 Ch.3, Lenses #2–6)

Schell builds up a working definition through a chain of provisional concepts: **A game is a problem-solving activity, approached with a playful attitude.** Not offered as canonical definition but as a thinking tool. Five lenses emerge (s005, ch.3).

## Detail

- **Toy** = object you play with. **Good toy** = fun to play with (satisfying to manipulate).
- **Fun** = pleasure with surprises (Lens #2: Surprise; Lens #3: Fun).
- **Play** = free movement within a more rigid structure (Salen & Zimmerman); Santayana: whatever is done spontaneously and for its own sake.
- **Game** = a problem-solving activity approached with a playful attitude. Games have goals, rules, and competition/challenge (Lens #6: Problem Solving). Salen & Zimmerman: games are a system where players engage in artificial conflict defined by rules resulting in a quantifiable outcome.
- **Endogenous value (Lens #5):** game objects and actions have value within the game context that they don't have outside it. The magic circle defines what is valuable.
- **Curiosity (Lens #4):** curiosity drives exploration and learning; games satisfy curiosity better than most media because the player can actively probe.
