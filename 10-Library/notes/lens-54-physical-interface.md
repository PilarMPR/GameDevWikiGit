# Lens #54 — Physical Interface

The physical input device shapes the game's feel fundamentally. Choose and design the physical interface to create the most direct, natural mapping to the game's core interactions. (s005, ch.13)

## Core Questions

> Is the physical input well-suited to this game? Can the mapping between input and game actions be more direct? Does the physical interface feel good?
