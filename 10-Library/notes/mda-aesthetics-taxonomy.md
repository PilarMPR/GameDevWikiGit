# MDA Aesthetics — 8 Kinds of Fun

LeBlanc, Hunicke, and Zubek replaced the vague word "fun" with a directed vocabulary of 8 aesthetic types that name specific emotional responses a game can target. Every deliberate design decision should be traceable to one or more of these aesthetics as its intended outcome (s011).

## Detail

| # | Aesthetic | Frame |
|---|---|---|
| 1 | **Sensation** | sense-pleasure: visual, audio, haptic, kinesthetic |
| 2 | **Fantasy** | make-believe, escapism, role-playing another self |
| 3 | **Narrative** | drama, story, character arc, unfolding events |
| 4 | **Challenge** | mastery, obstacle course, competition |
| 5 | **Fellowship** | social framework, cooperation, community |
| 6 | **Discovery** | uncharted territory, exploration, novelty |
| 7 | **Expression** | self-discovery, customization, creativity |
| 8 | **Submission** | pastime, low-stakes engagement, relaxation |

> ⚠️ Note: Schell lists 8 player pleasures (s005) using nearly identical framing — these two lists are functionally equivalent. See [pleasures-and-aesthetics-equivalence](pleasures-and-aesthetics-equivalence.md).
