# Lens #55 — Virtual Interface

The on-screen UI should give players the information they need without getting in the way. Good game UI is invisible when working; noticed only when it fails. (s005, ch.13)

## Core Questions

> Does the virtual interface give players the information they need? Does it ever get in the way? Is information placed where the player's attention already goes?
