# Lens #68 — The Hero's Journey

Campbell's monomyth: call → threshold crossing → trials → ordeal → transformation → return. The most resonant narrative structure across cultures. Not a formula but a pattern that speaks to deep psychological needs. (s005, ch.15)

## Core Questions

> Does the protagonist face a meaningful call to adventure? Is there a true threshold crossing? Are the trials meaningful and escalating? Is there genuine transformation?
