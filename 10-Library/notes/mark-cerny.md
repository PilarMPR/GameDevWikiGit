# Mark Cerny

Game designer and system architect. Known for Marble Madness, Crash Bandicoot (producer), Spyro, Jak and Daxter, PS4/PS5 lead system architect. Developed **Cerny's Method** for game development.

## Key ideas

- **Cerny's Method** — make a small, polished version of the core experience first (2 "publishable levels"); pre-production ends when those are genuinely fun. Production then expands the proven design. Cited by Schell (s005, ch.7).
- **The two publishable levels** — the pre-production deliverable that proves the concept; everything else is production scale-up.
