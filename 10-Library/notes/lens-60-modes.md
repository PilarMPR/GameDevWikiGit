# Lens #60 — Modes

When a player is in a mode, they need to know which mode they're in at all times. Mode confusion (I thought I was in combat, not in dialog) produces frustration and perceived unfairness. (s005, ch.13)

## Core Questions

> Are the modes of my game clear? Is it always obvious which mode the player is in? Can anything be done to reduce the number of modes?
