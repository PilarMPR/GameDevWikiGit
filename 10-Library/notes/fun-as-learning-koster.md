# Fun as Learning — Koster's Theory

Raph Koster's central thesis: **fun is just another word for learning.** The brain evolved to recognize patterns; games are stripped-down, compressed pattern-learning systems that deliver the pleasurable feedback of successful pattern recognition. When the pattern is mastered, the game is "used up" and becomes boring (s016).

## Detail

- **The brain craves patterns.** "Fun" is the feedback signal of the brain successfully learning/chunking a new pattern. The sensation of insight, discovery, and mastery is the game's core reward.
- **Boredom is the failure signal:** "Boredom is the brain casting about for new information — the pattern has been mastered." When a game stops teaching, it stops being fun (s016, ch.3).
- **The learnable middle:** too easy (pattern already known) = boring. Too hard (pattern looks like noise) = frustrating. Fun lives in between. This directly maps to the flow channel.
- **Definition of a good game:** "one that teaches everything it has to offer before the player stops playing." Depth = amount of novel pattern to learn before mastery.
- **Not all fun is learning-fun:** Koster distinguishes game-fun (pattern mastery) from aesthetic pleasure, social engagement, and storytelling — these can accompany a game but aren't its intrinsic fun. Games can carry meaning as an expressive medium (s016).

> ⚠️ Tension with s001: Sellers says a game need not be "fun" if it is highly interactive and engaging. Koster would say sustained engagement *is* ongoing learning. See [fun-as-engagement-sellers](fun-engagement-debate.md).
