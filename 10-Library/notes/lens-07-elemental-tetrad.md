# Lens #7 — Elemental Tetrad

Every game consists of four interdependent elements (Mechanics, Story, Aesthetics, Technology) that must be in harmony. Check that all four are considered and reinforce each other. (s005, ch.4)

## Core Questions

> Is my design using all four element types? Could it improve by enhancing one or more? Are the four in harmony, reinforcing each other toward a common theme?
