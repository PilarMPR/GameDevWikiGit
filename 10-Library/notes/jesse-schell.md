# Jesse Schell

Game designer, professor at Carnegie Mellon ETC, founder of Schell Games. Author of *The Art of Game Design: A Book of Lenses* (s005) — the most widely taught game design textbook; 100+ lenses across all aspects of design.

## Key ideas

- **Lenses methodology** — design is best practiced through 100+ perspective-shifting questions; each lens is a focusing tool, not a rule (s005).
- **Experience-first design** — the game is not the experience; design flows from target experience backward, not from features forward (s005, ch.2).
- **Elemental Tetrad** — Mechanics, Story, Aesthetics, Technology must reinforce each other (s005, ch.4).
- **Rule of the Loop** — the more you iterate, the better the game (s005, ch.7).
