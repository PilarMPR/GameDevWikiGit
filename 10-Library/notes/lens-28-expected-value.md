# Lens #28 — Expected Value

Probability × outcome = expected value. Good risk/reward decisions require comparable expected values between options. This is the mathematical foundation of triangularity. (s005, ch.10)

## Core Questions

> What are the outcomes and probabilities of each major decision? Are the expected values appropriate? Are risky choices worth the risk?
