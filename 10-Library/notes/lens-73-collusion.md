# Lens #73 — Collusion

NPCs and the environment should secretly cooperate to create the best possible experience. Characters who "notice" when the player needs help, environments that highlight the path — all are forms of collusion in service of the experience. (s005, ch.16)

## Core Questions

> Am I using collusion between my characters and environment? What would it look like if all elements secretly cooperated to create the best experience?
