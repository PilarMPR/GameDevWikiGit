# Lens #5 — Endogenous Value

Objects and actions in a game have value only within the game's magic circle. The designer must create that internal value — make players care about in-game achievements. (s005, ch.3)

## Core Questions

> Does my game give objects real value to the player? Do players feel their achievements matter within the game? How can in-game value be made more real?
