# Lens #51 — Pyramid

Start with simple versions of a mechanic; build toward complexity progressively. The Zelda dungeon design principle: introduce the item in a simple context, then progressively demand more sophisticated use. (s005, ch.12)

## Core Questions

> Does the challenge start simple and build to complexity? Is the difficulty curve gradual? Are easier versions of each challenge available before harder ones?
