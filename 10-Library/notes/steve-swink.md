# Steve Swink

Game designer. Author of *Game Feel: A Game Designer's Guide to Virtual Sensation* (s007).

## Key ideas

- **Game feel = real-time control of virtual objects in simulated space, with interactions emphasized by polish** (s007).
- **Game feel is measurable** — via input/response metrics (acceleration curves, latency, dead zones).
- **The 240ms boundary** — upper limit for real-time interaction feel.
- **Polish** = audio/visual effects that sell interactions (screen shake, hit-stop, sound sync).
