# Lens #79 — Character Web

Map the relationships between all characters. Relationships that change over time create narrative momentum. The web determines what dramatic interactions are possible. (s005, ch.18)

## Core Questions

> What are all the relationships between my characters? How do they change over time? Are there relationships that could generate more drama?
