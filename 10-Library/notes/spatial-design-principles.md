# Spatial Design — Organizing Game Space

Game spaces are functional systems that shape player behavior and emotion through architecture, sightlines, scale, and indirect control. Schell identifies five organizing patterns (linear → grid → web → points-in-space → divided spaces) from most to least controlled. Good spatial design begins with functional space (the floorplan) before experienced space (the art) (s005, ch.19).

## Detail

- **5 organizing patterns (s005, ch.19):**
  1. **Linear:** single path, maximum narrative control. Use for story-critical sequences.
  2. **Grid:** 2D map clarity; supports strategic planning. Use when player navigation aids challenge.
  3. **Web:** interconnected paths with real choice; supports exploration. Most action-adventures.
  4. **Points-in-space:** discrete locations in a free field. Open-world default. Requires dense landmarks.
  5. **Divided spaces:** sub-spaces with controlled connections. Default for most indoor games.
- **Functional vs. experienced space (s005):** functional space (topological structure — what connects to what) must be validated in gameplay first; experienced space (visual realization) built on top.
- **Sightlines:** the eye pulls toward contrast, motion, unique forms, and openings. Place goals at the end of sightlines. Balance tactical sightlines in multiplayer to avoid dominant positions.
- **Scale and emotion:** small spaces = claustrophobia, tension. Large spaces = awe, exposure. Sudden scale change creates powerful emotional response.
- **Spatial rhythm:** combat spaces (tension) → transitional areas (recovery) → reward spaces (release) = the interest curve implemented in 3D.
