# Michael Sellers

Game designer, entrepreneur, educator. Author of *Advanced Game Design: A Systems Approach* (s001).

## Key ideas

- **Games are systems; game design is system design** — the central thesis of his work (s001).
- **Parts, loops, and wholes** — every system decomposes into these three aspects (s001, ch.2).
- **Four principal loops** — game model, player mental model, interactive loop (core loop), designer's loop (s001, ch.7).
- **Engagement over fun** — games need not be "fun"; what matters is sustained engagement through the interactive loop (s001, ch.4).
