# Lens #76 — Character Function

Every character in a game should serve a clear functional purpose (narrative delivery, indirect control, challenge, emotional anchor). Characters serving no function are clutter. (s005, ch.18)

## Core Questions

> What function does each character serve? Are any characters redundant? Could characters serve multiple functions?
