# Lens #43 — Elegance

Measure elegance by how many purposes each element serves. Pac-Man's points serve 5 simultaneous purposes. Elements serving only one purpose are candidates for removal or redesign. (s005, ch.11)

## Core Questions

> Does each element serve multiple purposes? Are there elements that only serve a single purpose? How can I make each element serve more purposes?
