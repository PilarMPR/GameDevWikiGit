# Lens #69 — The Weirdest Thing

Every game has one weirdest thing that makes it unique. If there is too much weird, the game becomes incoherent. If no single element is weirdest, the game lacks identity. (s005, ch.15)

## Core Questions

> What is the weirdest thing in my game? Is there only one? Does it make the world more interesting and memorable?
