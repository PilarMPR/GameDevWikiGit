# Lens #39 — Time

Session length determines audience, social context, strategic depth possible, and narrative arc achievable. Match session length to intended use context. Provide graceful ending mechanisms. (s005, ch.11)

## Core Questions

> How long should the game last? Does time pressure add tension or just frustrate? Should the game be shorter, longer? Do players want to play again immediately after finishing?
