# Lens #61 — Interest Curve

Map the experience's interest level over time: hook → rising action → climax → resolution. The interest curve must stay above the threshold and have a fractal structure across all timescales. (s005, ch.14)

## Core Questions

> If I draw the interest curve of my experience, what shape is it? Does it have a hook? A climax? Does it cross the interest threshold? Is there fractal structure?
