# Lens #15 — The Toy

A good game starts with a good toy — a core interaction that is satisfying to play with even without goals. Build the toy first; add game structure once the fundamental interaction feels good. (s005, ch.7)

## Core Questions

> Is my game fun to play with without any goals? Does the core interaction invite manipulation for its own sake?
