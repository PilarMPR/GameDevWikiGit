# Game Analyses

Applying the wiki's frameworks to specific games. Each analysis includes formal elements, MDA analysis, core loop diagram, design highlights with concept links, and relevance to Hot Potato / educational games where applicable.

- [[jackbox]] — Jackbox Party Pack (2014–) — social loop architecture, player-generated content, phone input model

## Template

Each analysis follows the same structure:
1. Quick stats table (with Hot Potato/edugames relevance column)

3. MDA Analysis (s011)
4. Core loop diagram
5. Design highlights (each linked to a wiki concept page)
6. What it does well (extractable principles)
7. Relevance to Hot Potato or educational games
8. Related analyses
