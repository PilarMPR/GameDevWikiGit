# Art & Technical Art — Índice

- `art-direction.md` — style guides, consistency, referentes visuales
- `concept-to-3d-workflow.md` — concepto → modelado → rigging → engine

### Técnica UE5

- `material-system-ue5.md` — material instances, parameter collections, layering
- `vfx-niagara.md` — fundamentos de Niagara para efectos de gameplay
- `lighting-ue5.md` — Lumen, lightmaps, sky atmosphere, performance

### Personajes
- `character-rigging.md` — Control Rig UE5, retargeting, modular characters
- `animation-blueprint.md` — state machines de animación, blendspaces

### Optimización visual
- `texture-optimization.md` — formatos, compresión, streaming
- `draw-call-optimization.md` — instancing, HLODs, occlusion culling
- `performance-budget.md` — presupuestos de triángulos, draw calls, memoria por plataforma
