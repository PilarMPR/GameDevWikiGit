# Game Feel & the 3Cs

The moment-to-moment, kinesthetic layer of a game — how control *feels*.

## Game feel (Swink, s007)
[[../sources/s007-game-feel|Game Feel]] defines it as **real-time control of virtual objects in
a simulated space, with interactions emphasized by polish** (s007). Three building blocks:
1. **Real-time control** — responsive, continuous input in the perception–action loop.
2. **Simulated space** — physics/objects to navigate.
3. **Polish** — sound, particles, screen shake, animation that emphasize and "sell" interactions.

Game feel is **measurable** via input/response **metrics** (position, velocity, acceleration
curves), and is grounded in **perception** (proprioception, extending the self into the avatar) (s007).

## The 3Cs (s017)
[[../sources/s017-f2p-design-handbook|The F2P handbook]] frames the same layer as the **3Cs —
Character, Camera, Control** — the intuitive feel systems that must be nailed before deeper
systems (s017).
